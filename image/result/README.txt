This folder contains the image results for different face swapping and blending methods

direct_cut_source_swap = copy paste blending method + warping the source image to fit the face
poisson_blend_source_swap = poisson blend method + warping the source image to fit the face
direct_cut_target_swap = copy paste blending method + warping the target image to fit the face
poisson_blend_target_swap = poisson blend method + warping the target image to fit the face
